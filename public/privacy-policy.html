Politique de Confidentialité

Dernière mise à jour : Février 2026

Bienvenue sur Dames (ci-après "l'Application"). Nous accordons une grande importance à la protection de vos données personnelles. Cette politique explique comment nous collectons, utilisons et protégeons vos informations.

1. Données Collectées

Nous collectons les données suivantes :

  • Informations de compte : email, nom d'utilisateur, mot de passe (crypté)
  • Données de jeu : statistiques de parties, score ELO, succès débloqués
  • Données de transaction : historique des achats in-app (sans données bancaires)
  • Données techniques : type d'appareil, système d'exploitation, identifiants publicitaires

2. Utilisation des Données

Vos données sont utilisées pour :

  • Fournir et améliorer nos services de jeu
  • Gérer votre compte et vos achats
  • Afficher des publicités personnalisées (avec votre consentement)
  • Établir des classements et statistiques
  • Vous envoyer des notifications (bonus quotidien, tournois)
  • Prévenir la fraude et assurer la sécurité

3. Publicité

L'Application utilise Google AdMob pour afficher des publicités. AdMob peut collecter :

  • Identifiants publicitaires (IDFA/GAID)
  • Données d'interaction avec les publicités
  • Informations sur l'appareil

Vous pouvez désactiver la personnalisation des publicités dans les paramètres de votre appareil ou en souscrivant à l'offre Premium.

4. Partage des Données

Nous ne vendons jamais vos données personnelles. Nous les partageons uniquement avec :

  • Stripe : pour le traitement sécurisé des paiements
  • Google AdMob : pour l'affichage de publicités
  • Firebase : pour l'analyse et les notifications

5. Sécurité

Nous protégeons vos données par :

  • Chiffrement des mots de passe (bcrypt)
  • Communications sécurisées (HTTPS/TLS)
  • Accès restreint aux données
  • Serveurs sécurisés

6. Vos Droits (RGPD)

Conformément au RGPD, vous avez le droit de :

  • Accès : obtenir une copie de vos données
  • Rectification : corriger vos informations
  • Suppression : demander la suppression de votre compte
  • Portabilité : recevoir vos données dans un format standard
  • Opposition : refuser le traitement de vos données

Pour exercer ces droits, contactez-nous à : privacy@tccdames.com

7. Conservation des Données

Vos données sont conservées :

  • Tant que votre compte est actif
  • 3 ans après la dernière connexion (comptes inactifs)
  • Les données de paiement sont conservées selon les obligations légales (10 ans)

8. Mineurs

L'Application est destinée à un public de 12 ans et plus. Nous ne collectons pas sciemment de données sur les enfants de moins de 13 ans. Si vous êtes parent et pensez que votre enfant nous a fourni des informations, contactez-nous.

9. Modifications

Nous pouvons modifier cette politique. Les utilisateurs seront informés des changements significatifs par notification dans l'Application.

10. Contact
